# Image Metadata Editor

Un éditeur graphique pour visualiser, modifier et supprimer les métadonnées des images (JPEG, PNG, GIF, BMP).

## Fonctionnalités

- Affiche les informations de base du fichier et de l'image (nom, taille, dimensions, date de création/modification, permissions).
- Lit et affiche les métadonnées EXIF pour les JPEG.
- Affiche et modifie les informations additionnelles pour PNG.
- Supprime toutes les métadonnées si nécessaire.
- Interface graphique simple avec Tkinter.

## Formats supportés

- JPEG / JPG
- PNG
- GIF
- BMP
